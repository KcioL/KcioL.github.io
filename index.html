Projet Cook & Save

Contexte : Conception d'une application tablette pour une chaîne de restaurants fictive, visant à optimiser la gestion des stocks et la consommation énergétique.

  • Fonctionnalités : Suivi des stocks en temps réel et monitoring automatique des appareils (Frigos, Fours) via capteurs pour détecter les pannes.
  • Conception : Réalisation du cahier des charges, modélisation UML (Cas d'utilisation, Séquence) et maquettage (Wireframes) de l'interface.
  • Interface : Tableaux de bord pour les gérants (Suivi consommation) et interface simplifiée pour les employés (Entrées/Sorties stocks).
  • Objectif : Améliorer l'efficacité des restaurants et identifier les sources de gaspillage énergétique.

Stage TELERAD : Développement d'Application Statistique

Contexte : Développement d'un site web intranet pour le service Méthodes & Industrialisation. L'objectif était d'interroger la base de données de production pour générer automatiquement des statistiques d'aide à la décision.

  • Développement Full Stack : Conception de l'interface (HTML/CSS/JS) et de la logique serveur (PHP) pour connecter le site à la base de données Controle_process.
  • SQL & Optimisation : Rédaction et optimisation de requêtes SQL complexes (jointures, filtres dynamiques) pour extraire les données pertinentes.
  • Statistiques Industrielles : Intégration d'algorithmes en PHP pour calculer les indicateurs de capabilité (Cp, Cpk) et le rendement (FPY).
  • Visualisation de Données : Utilisation de la librairie Chart.js pour créer des graphiques interactifs (Droites de Henry, Courbes de Gauss, Histogrammes).
